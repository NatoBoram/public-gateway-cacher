Public IPFS Gateways

About

This is a fork of github.com/ipfs/public-gateway-checker that allows you to cache a specific IPFS and IPNS hash to a bunch of public gateways. The code source is available at gitlab.com/NatoBoram/public-gateway-checker.

If you'd like to add a new public gateway, please go to github.com/ipfs/public-gateway-checker, submit a pull request then open an issue here.
